h2. About

These are a set of commands for Mozilla Ubiquity that interact with Read It Later's simple API to let you post new pages to you Read It Later list. That's it.

h2. Commands

* readlater.auth _username_:_password_
** Saves your username and password to Firefox's preferences
* readlater
** Add current page to your Read It Later list.
